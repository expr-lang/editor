# Expr Editor

> [!IMPORTANT]
> This is a work in progress.

Embeddable expression editor for [Expr](https://expr-lang.org).

* Supports React, Vue, Angular, and vanilla JS.
* TypeScript and JavaScript.
* Supports custom functions and variables.
